# 1. What is Programming?

## 📖 Definition

Programming is the process of writing instructions (called code) that tell a computer what to do.

---

## 🌍 Real-Life Analogy

Imagine making a cup of tea.

Steps:

1. Boil water
2. Add tea powder
3. Add milk
4. Add sugar
5. Pour into a cup

These are instructions.

Programming works the same way.

Instead of giving instructions to a person, we give instructions to a computer.

---

## Example

```python
print("Hello, Sathiya Sri!")
```

Output

```
Hello, Sathiya Sri!
```

The computer follows your instruction exactly.

# 3. What is Python?

It is widely used in:

- Artificial Intelligence
- Machine Learning
- Data Science
- Web Development
- Automation

Released in 1991.

# 7. How Python Executes Code

```
Python Code
      ↓
Python Interpreter
      ↓
Reads one line
      ↓
Executes one line
      ↓
Displays Output
```

Python is called an **interpreted language** because it executes code one line at a time.

# 10. Comments

Comments are notes for programmers.

Python ignores comments.

Example

```python
# My first Python program

print("Hello")
```

Shortcut in VS Code

```
Ctrl + /
```

---

# 11. Python Syntax Rules

- Python is case-sensitive.
- Strings must be inside quotes.
- Use indentation properly.
- One statement per line.

Correct

```python
print("Python")
```

Wrong

```python
Print("Python")
```
