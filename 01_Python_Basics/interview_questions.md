# Day 1 - Interview Questions

## 1. What is Programming?

### Answer

Programming is the process of writing instructions (code) that tell a computer what to do.

---

## 2. What is a Programming Language?

### Answer

A programming language is a language used to communicate with computers and write programs.

Examples:
- Python
- Java
- C
- C++
- JavaScript

---

## 3. What is Python?

### Answer

Python is a high-level, interpreted, general-purpose programming language.

---

## 4. Who created Python?

### Answer

Guido van Rossum.

---

## 5. When was Python released?

### Answer

1991.

---

## 6. Why is Python popular?

### Answer

- Easy syntax
- Readable
- Large libraries
- AI & Data Science
- Cross-platform
- Open source

---

## 7. What is an Interpreter?

### Answer

An interpreter reads and executes Python code one line at a time.

---

## 8. What does print() do?

### Answer

The `print()` function displays output on the screen.

Example:

```python
print("Hello")
```

Output:

```
Hello
```

---

## 9. What are Comments?

### Answer

Comments are notes written for programmers.

Python ignores comments during execution.

Example:

```python
# This is a comment
```

---

## 10. Difference between Programming and Python?

### Answer

Programming is the process of writing instructions.

Python is a programming language used to write those instructions.
